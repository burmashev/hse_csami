Problem ku06-2: kr06-2 (дореш)

Напишите функцию void myfloorf(float *arg), которая округляет число *arg вниз к ближайшему целому и записывает результат обратно в *arg. Гарантируется, что в момент вызова myfloorf значение *arg – положительное нормализованное число, не бесконечность и не NaN.

Используйте операции с целыми числами (битовые маски и т. д.) Не используйте операции с вещественными числами, в том числе преобразование типов (int i = (int)*arg).

Решение будет тестироваться с флагами компилятора -m32 -msoft-float -mfpmath=387, так что программы, использующие операции с вещественными числами, не скомпилируются.

Examples

Input

1.5

Output

1.0

Input

0.1

Output

0.0

Input

15.43

Output

15.0
