Problem ku01-3: kr01-3 (дореш)

Напишите подпрограмму mul2, которая принимает один параметр — адрес 32-битного знаковой переменной value. Подпрограмма должна умножить value на 2, если при этом не возникнет переполнения. Если переполнение возникло бы, то значение value должно остаться неизменным.

(Подпрограмма соответствует объявлению void mul2(int *value) на языке Си и, если не происходит переполнения, меняет значение в памяти по переданному адресу.)
